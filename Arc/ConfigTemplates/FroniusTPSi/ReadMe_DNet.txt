Configuration information for Fronius TPS/i welder

Default IO board address
Robtot 1	-DN_Address 40
Robtot 2	-DN_Address 41
Robtot 3	-DN_Address 42
Robtot 4	-DN_Address 43

Default WireFeed unit is mm/s
Unit can be changed by modifying the scaling on AI aiFr1WireFeed_M
By adding or replace one of the following cfg files, the scaling can be modified
SetWF_to_MPM_ROB1.cfg, scaling will be set to m/min for robot 1
SetWF_to_IPM_ROB1.cfg, scaling will be set to Inches/min for robot 1
There are also files for Robot 2, 3 and 4.
